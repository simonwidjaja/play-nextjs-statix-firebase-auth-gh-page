Introduction

Hello and welcome to this short introduction into the next level of authoring web content in Adobe Animate CC.  In the following chapters you will learn what the Toolbelt extension for Animate is and how to use it. You will also get to know the different features with hands-on examples.

If you want to be informed about new features and updates, make sure to sign up on richcontentdesign.com or follow us on twitter (@designlab.io)

What is the Toolbelt?

The Toolbelt for Animate CC is an external JavaScript library that allows you to add animations and functionality without having to use a timeline or code. Note that this extension only works for HTML5 Canvas projects.

How does the Toolbelt work?

The Toolbelt is designed to be as easy to use as possible - without having to write code or create complex animation tweens - while maintaining a high level of flexibility and expressiveness. The approach followed here is “convention over implementation”. For you as a user, that means you only have to follow two easy steps:

  • Include the extension library in your project

  • Name your movieclip or text field instance accordingly

  • Define the Toolbelt feature with initial letters, e.g. TA for typo animation

  • Add optional parameters like type, delay, etc. (“TA_TYPE_SCRAMBLE_DELAY_4500”) That’s all.

Why would I use the Toolbelt?

The Toolbelt is designed to do tedious or code savvy work tasks in Animate for you. That means a higher level of productivity and - in some cases - enablement as it allows you to do things you would normally not be able to do in Animate.

Let’s take a look at some examples:

  • Typo Animations can be created with the timeline in Animate. However, having to break down the text elements into single letters for creating the effects deprives you of the option to edit the text as a whole. Here using the Toolbelt makes it way more convenient (the text remains whole during the authoring process) and much faster (you only have to name the layer instead of creating the entire animations manually).
  • Hotspot Overlay ToggleFor an overlay to appear and disappear when the user clicks on a hotspot, it requires quite a bit of logic and therefore code. While this is not too difficult for someone well versed in JavaScript, for most Animate users this is at least a challenge. With the Toolbelt you don’t have to worry about code. Just name the Hotspot movieclip and the Overlay movieclip accordingly and you’re done.
  • Language localization and Line Drawing These are features that allow you to do things in Animate you would otherwise not be able to do in such a way. While you can use dynamic text fields for changing the content of a text field during run time, being able to simply add JSON files with translated text next to the project file to automatically show the right content depending on the selected language is a whole new level. And with form tweens you can create straight line animations, but no real complex line drawing animations.

The Toolbelt, in short, lets you work faster, more convenient and gives you creative superpowers.

What can I do with the Toolbelt?

The Toolbelt comes with a set of different features. Each feature can have different options. For example one of the current features is Typo Animations, which has for instance the option to let the text appear as single letters.

Future updates of the Toolbelt can then have completely new features and/or new options for existing features.

Here is an overview of the features and options of the current version:

  • Typo Animations

  • Appear

  • Typewriter

  • Scramble

  • Line drawing

  • Localization (dynamically showing text in a specific language)

  • Toggle Hotspot Overlay

… and more to come!
